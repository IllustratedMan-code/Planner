How do you write a joint probability table?
** Answer
Create a table of the following form
|                   | C = 0 | C = 1 |
| /                 | <>    |       |
|-------------------+-------+-------|
| X = 1, Y = 1, etc |       |       |
| All combinations  |       |       |
